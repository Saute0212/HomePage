# HomePage
This is the home page.

# ディレクトリ構造
```bash
/HomePage
│── /assets           # 静的ファイル（画像・フォントなど）
│   ├── /images       # 画像ファイル
│   ├── /fonts        # Webフォント
│   ├── /icons        # アイコン
│── /css              # スタイルシート
│   ├── styles.css    # メインのCSS
│   ├── reset.css     # リセットCSS
│── /js               # JavaScript
│   ├── script.js     # メインのJavaScript
│── /news             # お知らせ記事を格納
│   ├── 2025-03-01-new-feature.html # 記事ごとにHTMLファイルを作成
│   ├── 2025-02-15-maintenance.html
│   ├── index.html    # お知らせ一覧ページ
│── /pages            # 固定ページ（会社情報、サービス、問い合わせなど）
│   ├── about.html    # 会社概要
│   ├── services.html # サービス紹介
│   ├── contact.html  # 問い合わせ
│── index.html        # トップページ
│── README.md         # プロジェクトの説明
│── .gitignore        # Git管理しないファイルのリスト
```
